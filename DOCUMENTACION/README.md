*# 📘 Sistema de Gestión de Procedimientos*



*## 📌 Descripción*

*Sistema web desarrollado en PHP y MySQL que permite la gestión de procedimientos internos dentro de una organización, incluyendo control de usuarios, roles y reportes.*



*---*



*## 🎯 Objetivo*

*Facilitar el acceso, consulta y mejora continua de procedimientos operativos mediante una plataforma digital segura.*



*---*



*## 🧩 Funcionalidades*



*### 👤 Usuarios*

*- Registro con verificación por correo*

*- Inicio de sesión seguro*

*- Recuperación de contraseña*



*### 🔐 Roles*

*- \*\*Admin\*\**

&#x20; *- Gestión de usuarios*

&#x20; *- Crear procedimientos*

&#x20; *- Ver reportes*



*- \*\*Supervisor\*\**

&#x20; *- Crear procedimientos*

&#x20; *- Ver reportes*



*- \*\*Usuario\*\**

&#x20; *- Consultar procedimientos*

&#x20; *- Reportar errores o mejoras*



*---*



*### 📄 Procedimientos*

*- Creación por pasos*

*- Imágenes por paso*

*- Filtrado por área y máquina*



*---*



*### 🚨 Reportes*

*- Reportar errores o mejoras en procedimientos*

*- Notificaciones para administradores y supervisores*



*---*



*## 🛡️ Seguridad Implementada*



*- Encriptación de contraseñas (`password\_hash`)*

*- Validación de sesiones*

*- Control de acceso por roles (RBAC)*

*- Protección de rutas (`auth.php`)*

*- Recuperación de contraseña mediante token*



*---*



*## 🗄️ Base de Datos*



*Tablas principales:*



*- `users`*

*- `addprocedimientos`*

*- `procedimiento\_pasos`*

*- `paso\_imagenes`*

*- `reportes\_procedimientos`*

*- `password\_resets`*
